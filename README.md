Projeto desenvolvido para automatização de um quarto. O Master será o responsavel em receber as requisições e ordenar outros modulos (Slaves) a fazer determinada ação. Ele também possuirá alguns mecanismos de tomadas de decisões sozinho, sem precisar que o usuario mande uma requisição para fazer. A configuração e o envio das requisições do sistema será feito por meio de um aplicativo desenvolvido para iOS nativo (Swift).

Bibliotecas utilizadas desenvolvidas pela comunidade (algumas modificadas):
	-> Ethernet_v2
	-> RESTserver
